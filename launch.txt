Pour faire tourner le backend (le collecteur) :

Ouvrez un terminal.

Allez à la racine du projet : cd D:\coding\job_alert.

Activez l'environnement Python : .\.venv-stable\Scripts\Activate.ps1.

Lancez le script : python collector.py.

Pour faire tourner le frontend (l'interface web) :

Ouvrez un DEUXIÈME terminal (c'est plus pratique).

Allez à la racine du projet : cd D:\coding\job_alert.

Déplacez-vous dans le dossier du frontend : cd ui.

Pas besoin d'activer le venv ici. Lancez directement : npm run dev.
